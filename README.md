# 🎉 DreamParty

DreamParty is an AI-powered birthday planning platform that helps users create personalized celebrations, beautiful invitations, and memorable party experiences.

Users can generate themed birthday ideas, receive AI-powered recommendations, explore zodiac-inspired inspiration, organize party details, and create customized invitation cards.

## ✨ Features

- Personalized birthday invitations
- AI-powered theme recommendations
- Zodiac-inspired celebration ideas
- Color palette suggestions
- Party planning tools
- Guest management
- Smart celebration inspiration

## 🚀 Project Vision

DreamParty aims to make birthday planning simple, creative, and personalized through modern web technologies and AI-powered assistance.

Users can transform a simple birthday idea into a complete celebration experience with themes, invitations, planning tools, and inspiration.
